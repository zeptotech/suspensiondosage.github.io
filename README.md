# Vibe coded Antibiotic Suspension Dosage Calculator
Lightweight, intuitive client-side web tool for calculating dose (and reverse dose calculation) for two major antibiotic suspensions. Vibe-coded using Claude.

## Drugs included
- Amoxicillin oral suspension
- Cephalexin oral suspension

## Features
- Enter weight in kg or lb
- Enter dose in mL or mg
- Ability to choose between commercially available concentrations
- Select dosing frequency
- Calculate mg/kg/day
- Reverse dosing capability as well
- Fully client side, no data leaves the browser
